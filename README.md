# JPDB Hotkeys

Keyboard shortcuts for [jpdb.io](https://jpdb.io) review page to improve your Japanese learning efficiency.

## Features

- **Keyboard Navigation**: Use WASD or HJKL (Vim-style) keys to grade reviews
- **Fast Grading**: Press keys to grade without reaching for the mouse
- **Auto-show Examples**: Automatically toggles examples on for better context
- **Kanji Blacklist**: Auto-blacklist kanji cards (SVG-based)

## Installation

1. Install a userscript manager:
   - [Violentmonkey](https://violentmonkey.github.io/) (Recommended)
   - [Tampermonkey](https://www.tampermonkey.net/)
   - [Greasemonkey](https://www.greasespot.net/)
   - [ScriptCat](https://scriptcat.org/)

2. Install this script:
   - Click here: [jpdb-hotkeys.user.js](https://github.com/snomiao/jpdb-hotkeys.user.js/raw/main/jpdb-hotkeys.user.js)
   - Your userscript manager should prompt you to install

## Hotkeys

| Keys | Action |
|------|--------|
| <kbd>H</kbd> or <kbd>A</kbd> | Grade as "Easy" (I know this) |
| <kbd>J</kbd> or <kbd>S</kbd> | Grade as "Hard" (but may forget) |
| <kbd>L</kbd> or <kbd>D</kbd> | Grade as "Nothing" (don't know) |
| <kbd>5</kbd>, <kbd>T</kbd>, or <kbd>;</kbd> | Blacklist card |

### Grading Guide

- **Easy (H/A)**: You know this word well and can recall it easily
- **Hard (J/S)**: You know it but might forget - needs more practice
- **Nothing (L/D)**: You don't know this word

## Usage

1. Navigate to any jpdb.io review page (e.g., https://jpdb.io/review)
2. Start reviewing as normal
3. Use the hotkeys to grade cards quickly
4. Press the hotkeys again to continue to the next card

## Compatibility

- Tested on jpdb.io review pages
- Works with all major browsers (Chrome, Firefox, Edge, Safari)
- Requires a userscript manager extension
